# 七、审核逻辑（你记住一句话）

微信审核看的是：

**是不是电商行为**

不是看你是不是卖衣服。

所以只要：

* 不出现 **购买**
* 不出现 **价格**
* 不出现 **购物车**
* 不出现 **商品**

就基本没问题。
